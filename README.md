## 🌟 About the Event

**Build with AI 2025** is a premier developer event hosted by **[GDG Assiut](https://www.facebook.com/GDGAssiut)**, part of a global initiative by Google for Developers. The event is designed to enable developers, tech enthusiasts, and problem-solvers to learn, apply, and build cutting-edge applications using the latest AI and Machine Learning technologies.

From Generative AI (Gemini, Gemma) to Vertex AI and practical agent-building, this event brought together the brightest minds in Assiut to explore, hack, and shape the future of technology together.

## 📂 About This Repository

Welcome to the official repository for **Build with AI - GDG Assiut 2025**! 

We created this space to serve as the central hub for our community. Here, you will find:
- 📝 **Session Materials:** Presentation slides, PDF references, and useful links shared by our expert speakers.
- 💻 **Source Code & Labs:** Starter codes, examples, and notebooks used during the hands-on technical workshops.
- 🚀 **Community Projects:** A showcase of the awesome AI applications built by attendees during and after the event.
- 🛠️ **Resources:** Handouts, prompt engineering cheat sheets, and recommended reading.
